# Copilot Plugin Marketplace

A central hub for discovering, sharing, and iterating on [GitHub Copilot](https://github.com/features/copilot) plugins and agentic coding infrastructure.

## Overview

This repository is a community-driven marketplace for Copilot plugins — a free, open resource for developers building and sharing agentic coding tools. It serves as the central spot for iterating on agentic coding infrastructure.

## Getting Started

Browse the available plugins, contribute your own, or use this repo as a starting point for building your own Copilot plugin ecosystem.
