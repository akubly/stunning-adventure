# @akubly/crucible-cli

The `crucible-cli` package is the command-line shell for the **Crucible agentic runtime** — a 5-layer append-only session engine featuring deterministic replay, counterfactual branching, and per-row safety invariants. It exposes user-facing commands (`fork`, `replay`, `bisect`, etc.) that drive the Crucible runtime layers (L0–L5) described in the [Crucible Technical Design](../../docs/crucible-technical-design/).
